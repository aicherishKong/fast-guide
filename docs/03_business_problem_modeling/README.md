## 核心指标体系（草稿）

```
北极星指标：用户收到面试邀约率（约面率）
  ├── 简历维度
  │     ├── 简历被查看率
  │     ├── JD 关键词匹配分
  │     └── 简历改写采纳率
  ├── 面试维度
  │     ├── 模拟面试完成率
  │     ├── 面试表现分提升幅度
  │     └── 二面通过率
  └── 学习维度
        ├── 知识训练完成率
        ├── 费曼总结通过率
        └── 复测提升分
```
